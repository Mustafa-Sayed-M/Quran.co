# Quran.co 🕌

**Quran.co** is a modern web application built using **React 19**, **Vite**, and **Tailwind CSS**.  
The app allows users to read and interact with Quranic verses, listen to recitations, and explore Hadiths and Tafsir in a clean UI.

---

## 🚀 Features

- 📖 Browse Quranic chapters and verses
- 🎧 Play verse recitations
- 💾 Save favorite verses
- 🗂️ Organized UI with modals and audio context
- ⚙️ Built with React Context API and React Router
- 🌙 Dark Mode & other settings coming soon

---
