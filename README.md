# 🌙 Quran.co

**Quran.co** is a modern and user-friendly web application that allows users to explore the Quran with ease. It provides access to all surahs, audio recitations, translations, and tafsir (interpretation).

## 🧩 Features

- 📖 Full list of Quranic surahs
- 🔊 Audio recitation for each surah
- 🌍 Multi-language translation support
- 📚 Tafsir (interpretation) for deeper understanding
- 🌙 Clean, responsive, and minimal UI
- ⚡ Built with performance and accessibility in mind

## 🚀 Tech Stack

- **React.js** – Frontend framework
- **Tailwind CSS** – For fast and modern UI styling
- **Quran API** – For fetching Quranic data and recitations
- **React Router** – Page routing

## 🛠️ Installation

```bash
git clone https://github.com/your-username/quran.co.git
cd quran.co
npm install
npm start
```

## 📡 API Used

We use the Quran API to retrieve Quranic data.
